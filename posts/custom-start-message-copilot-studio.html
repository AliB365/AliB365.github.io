Personalise Your AI: Changing the Start Message in Copilot Studio

November 16, 2025 5 min read Autonomous Agentic
✨ Why change the start message?
The first message your agent sends sets the tone for every conversation. Make it memorable, helpful, and on-brand to set a good opener for your users!

Introduction

Copilot Studio has hidden the ability to change the agent's start message, but it is actually very easy to do. After this short article, you will be able to change your agent's starter message with ease.

Main section

When you create a new agent, whether from scratch or using Copilot Studio, it will always have the same starter message:

Hello, I'm Copilot Studio Demo, a virtual assistant. Just so you are aware, I sometimes use AI to answer your questions. If you provided a website during creation, try asking me about it! Next try giving me some more knowledge by setting up generative AI.

Below, I will show, step by step, how to change the agent's “Conversation start”.

💡 Pro Tip: Personalize your start message with your agent's name, a friendly greeting, or a quick tip about what it can do!
  1. Open your agent in Copilot Studio
    Go to Copilot Studio and open the agent you want to edit.
  2. Navigate to the 'Topics' tab
    At the top navigation, click on the 'Topics' tab to access the conversation topics for your agent.
  3. Open the 'Conversation start' topic
    Next to the 'Custom' selected filter button, click the 'System' button. This will display all the predefined system topics, and you should see one called 'Conversation Start'. This is the topic that contains the starter message.
  4. Edit the start message
    Click on the 'Conversation Start' topic to open it. To edit the message just click on the text in the 'Message' box. You can edit this message to whatever you like.
  5. Save your changes
    Once you have edited the message, make sure to click the 'Save' button at the top right to save your changes.
🎉 That's it! Your agent will now greet users with your custom message. Try chatting with it to see the new experience in action.
