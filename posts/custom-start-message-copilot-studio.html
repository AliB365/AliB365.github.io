Personalise Your AI: Changing the Start Message in Copilot Studio

November 16, 2025 5 min read Autonomous Agentic
🎯 What you'll achieve:
In just a few minutes, you'll learn how to customise your agent's first impression by changing the start message. Make your agent feel unique, on-brand, and instantly helpful from the very first interaction.

Introduction

First impressions matter, especially for AI agents! The start message is the first thing users see when they begin a conversation with your agent, setting expectations and establishing the tone for the entire interaction. While Copilot Studio makes this customisation easy, the option is somewhat hidden in the system topics.

By default, every new agent greets users with a generic message that doesn't reflect your brand or the agent's specific purpose. In this guide, you'll discover exactly where to find and customise this message, transforming that generic greeting into something meaningful and memorable for your users.

Understanding the Default Start Message

When you create a new agent in Copilot Studio, whether from scratch or using a template, it comes with a standard greeting:

Hello, I'm Copilot Studio Demo, a virtual assistant. Just so you are aware, I sometimes use AI to answer your questions. If you provided a website during creation, try asking me about it! Next try giving me some more knowledge by setting up generative AI.

This message is functional but impersonal. It doesn't tell users what your agent specialises in, what problems it can solve, or how it can help them specifically. Let's change that!

💡 Pro Tip: When crafting your start message, include your agent's name, a warm greeting, and a brief description of what it can help with. For example: "Hi! I'm Autonomous Agentic, your HR assistant. I can help you with benefits questions, time-off requests, and company policy information. What can I help you with today?"

Customising Your Agent's Start Message

Follow these simple steps to create a start message that reflects your agent's personality and purpose. The process involves navigating to a system topic that controls the conversation start behavior.

  1. Open Your Agent in Copilot Studio
    Navigate to copilotstudio.microsoft.com and select the agent you want to customise from your dashboard. You'll land on the agent's overview page, which shows key metrics and quick access to configuration options.

    This overview page is your home base for managing all aspects of your agent, from testing to deployment.
  2. Navigate to the Topics Section
    In the top navigation bar, click on the 'Topics' tab. Topics are the building blocks of your agent's conversational ability, each topic handles a specific type of user request or interaction.

    The Topics page displays all the conversation flows you've created, but we're looking for something more fundamental: the system topics that control core behaviors.
  3. Access System Topics
    By default, you'll see your custom topics. To access the built-in system topics, look for the filter options near the top of the page. Next to the 'Custom' filter button, click the 'System' button.

    This reveals all the predefined system topics that control essential agent behaviors. You'll see several system topics, but the one we want is called 'Conversation Start'this is where the magic happens! System topics are powerful because they control fundamental aspects of how your agent behaves.
  4. Edit the Start Message
    Click on the 'Conversation Start' topic to open its editor. You'll see the conversation flow, which includes a 'Message' node containing the default greeting text.

    Simply click directly on the text inside the 'Message' box to start editing. You can completely replace the default message with your custom greeting. Consider including:
    • A personalised greeting with your agent's name
    • A brief description of what the agent can help with
    • An invitation to ask questions or start a task
    • Your brand's tone and personality

    You can also add formatting, emojis, or multiple lines to make your message more engaging and readable.
  5. Save Your Changes
    Once you're happy with your new start message, click the 'Save' button in the top right corner of the page. This commits your changes to the agent.

    Your changes are saved but not yet live, you'll need to publish your agent for users to see the new greeting. You can test it immediately using the built-in test chat to see how it looks before publishing to production.

Best Practices for Start Messages

Now that you know how to change the start message, here are some tips to make it as effective as possible:

  • Keep it concise: Users should be able to read and understand your message in 3-5 seconds
  • Be specific: Tell users exactly what your agent can do rather than using vague language
  • Set expectations: Let users know if there are topics the agent can't help with
  • Use friendly language: Write like a helpful colleague, not a formal manual
  • Include a call to action: Invite users to ask a question or try a specific feature
  • Test with real users: Get feedback on whether your message is clear and welcoming
🎉 You Did It!
Your agent now has a personalised start message that reflects its purpose and your brand. This small change makes a big difference in user experience, users will immediately understand what your agent can do and feel more confident engaging with it. Test it out in the chat window and see the difference for yourself!
